## Executive Summary
- **Objective**: Identify **top 20 existing drugs** with plausible **mechanistic relevance to acute myeloid leukemia (AML)** (with emphasis on **MOLM13 proliferation inhibition**) using a **knowledge-graph (KG) repurposing approach**, and provide **experimentally testable IC50 assay concentration series**.
- **Approach**: Built an AML KG (EFO_0000222) integrating **disease→gene/protein→pathway/MoA→drug** relationships; expanded to drugs connected to AML-associated **proteins, pathways, and mechanisms of action**; merged and **filtered out KG-known AML drugs** to focus on repurposing novelty; ran a battery of **ADMET predictions**; computed **mechanistic-first ranking** (ADMET secondary); generated visualizations; produced **MOLM13-ready 10-point dilution series** for the top 20 novel candidates.
- **Key Results**:
  1. Constructed AML KG with **5,314 nodes / 11,712 edges** and extracted **40 proteins**, **20 pathways**, **67 MoAs**, **80 KG-known AML drugs**.
  2. Assembled a **master candidate list of 563 unique drugs**, then created a **novel set of 483** by filtering out the 80 KG-known AML drugs.
  3. Generated **master_ranked** (all compounds) and **master_ranked_novel** (novel-only), with **mechanistic score as the primary ranking factor**.
  4. Produced **top 20 novel repurposing candidates** and **MOLM13 IC50 assay concentration series** (10-point, 3-fold).
- **Recommendations**:
  - Start with the **Top 20 novel list** from `top20_candidates.csv` and run **CellTiter-Glo (or equivalent) viability IC50 assays in MOLM13** using the provided dilution series.
  - For more potency-optimized dosing, add a follow-up step to retrieve **compound-specific IC50/Ki** from **ChEMBL** and tune top concentrations (e.g., 30–100× target IC50, capped).
  - Use ADMET predictions only as **secondary deprioritization** (e.g., strong hERG/AMES flags) rather than primary selection, consistent with the workflow design.

## Workflow Overview
- **Task Breakdown**
  1. **KG creation & querying** (proteins, pathways, MoAs, drugs; then drugs linked to all entities)
  2. **Data merging + filtering** (remove KG-known AML drugs to accelerate novel discovery)
  3. **ADMET prediction** (secondary ranking factor)
  4. **Integrated scoring + ranking + visualization** (mechanistic score primary)
  5. **IC50 assay concentration generation** (placed after ranking, as requested)
  6. **Reporting**

- **Agent Coordination**
  - **Research Agent**: KG creation and entity/drug extraction; generated entity-to-drug mapping files.
  - **Data Agent**: Merged datasets; filtered known AML drugs; engineered mechanistic features; consolidated ADMET outputs; computed rankings; generated figures; produced assay concentrations.
  - **Prediction Agent**: Executed ADMET models (CYPs, hERG, AMES, PGP, PAMPA, BBB, solubility, lipophilicity).

- **Process Flow**
  - AML → KG → (proteins/pathways/MoAs/drugs) → expand drugs for entities → merge & de-duplicate → filter known AML drugs → ADMET predictions → mechanistic-first rank → visualize → output top 20 + IC50 assay series.

## Integration and Synthesis: combining and interpreting findings
- **Cross-Agent Insights**
  - Research Agent provided entity/drug networks; Data Agent quantified evidence strength via **link counts** and **entity breadth**; Prediction Agent contributed **secondary risk/permeability/solubility indicators**.

- **Validation**
  - Internal cross-check: top candidates are those with high **KG linkage density** (protein/pathway/MoA coverage). ADMET used as secondary filter, not driver.

- **Limitations**
  1. **No numeric bioactivity (IC50/Ki) values** were present in the exported KG mapping tables, so assay concentrations could not be potency-calibrated from ChEMBL within this run.
  2. The “known AML drug” filter is based on membership in `known_drugs.csv` (KG-derived). Some AML-used but not explicitly captured drugs may remain.
  3. “MOLM13-specific” efficacy is not directly modeled; it is inferred mechanistically. Wet-lab validation is required.

- **Scientific Significance**
